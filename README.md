# BDA_IoT-based-Virtual-Health-Determination
本研究主要透過機器手臂的三軸振動訊號資料建立穩健且可靠的機器學習模型，以有效降低人工巡場所需的時間和人力成本，同時能夠及時精確地發現機器運作中的異常狀況。我們藉由時域與頻域特徵分析（如RMS、頻率中心、峭度、功率譜密度等）識別可能的磨損、鬆動或結構損傷並運用卷積神經網路（CNN）和隨機森林 (RF) 等機器學習方法建立預測模型，透過多層次的數據分析提升預測準確度。此外，我們設計直覺且易於理解的儀表板介面，將複雜的數據轉化為易讀資訊，讓現場操作人員能夠快速判讀機器狀態並做出適當的維護決策。
